# Sittax ST — Landing Page de Lançamento

Landing page de lançamento do **Sittax ST**, sistema de apuração automatizada de ICMS-ST, DIFAL e DIFAL Revenda para escritórios contábeis.

## Estrutura

| Arquivo | Descrição |
|---|---|
| `index.html` | Página principal |
| `style.css` | Estilos da landing page |
| `app.js` | Scripts (countdown, FAQ, scroll) |
| `config.js` | Configurações editáveis |

## Dobras da página

1. **Hero** — Lançamento, countdown, formulário, bônus de CNPJs
2. **Vídeo** — Tour guiado do produto
3. **Features** — Importação, classificação, identificação automática
4. **Regimes** — Simples Nacional, Lucro Presumido, Lucro Real
5. **Comparativo** — Antes e depois com Sittax ST
6. **FAQ** — 5 perguntas e respostas com accordion
7. **Ecossistema** — Sittax Simples + Sittax Recupera
8. **Footer** — Logo, texto institucional, ReclameAQUI, dados legais

## Configurações rápidas

Edite o arquivo `config.js` para alterar:
- Data de encerramento do countdown
- Número de vagas preenchidas
- ID do vídeo do YouTube
- Links UTM dos produtos
